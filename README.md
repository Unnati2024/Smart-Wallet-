# Smart-Wallet-
Smart Wallet is a Java-based CLI application designed to simplify expense splitting and debt settlement among individuals or groups. It allows users to record expenses, categorize them, convert currencies, track balances, and settle debts efficiently. The project focuses on applying core Java and data structures to solve a real-world financial problem in a clean, structured way.

The application supports multi-currency expenses, maintains a detailed transaction history, provides search and undo functionality, and ensures accurate balance computation using optimized data structures.

This project demonstrates strong fundamentals in Java, OOP principles, and data structures while emphasizing scalability, clarity, and usability through a command-line interface.
Data Structures Used:
ArrayList-->	Store expenses and transaction records
HashMap-->	Track user balances and category-wise expenses
Queue	-->Manage pending debt settlements
Stack-->	Implement undo functionality
UUID-->	Generate unique transaction IDs
